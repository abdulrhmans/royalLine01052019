Contact Privileges By Tags

This module was created to give you the ability to control the contacts a user can access. It enables you to add Contact Tags on the user form which will make the user only access contacts with the specified tags.

User Guide

Create tags for your contacts from contacts configuration (while developer mode is active).

Go to contacts and assign one or more tags to your contacts (contacts with no specified tags can be accessed by all users).

Create a user, fill in their details and access rights and check the 'Limited Contacts' box so that the 'Contacts Access' field & 'Tagged Allowed Contacts' tab would appear then save.

Assign certain tags per user which limits their access.

You can also check which contacts the user can access from the 'Tagged Allowed Contacts' tab.

Log in as a user and go to contacts and you will only be able to access certain contacts (notice the contacts viewed contain the tags assigned to this specific user in the previous steps).

The user also can't access any partner related documents to which they don't have access (such as SO's) . They can only access the documents of the contacts they have privileges to.
